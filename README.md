# Election Survey Data Analysis

This project analyzes election survey data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Steps
1. Loaded survey dataset using Pandas
2. Performed data cleaning and preprocessing
3. Checked missing values and data types
4. Conducted exploratory data analysis (EDA)
5. Analyzed voter eligibility, gender distribution, and voting preference
6. Created visualizations using Matplotlib

## Insights
- Voter age distribution
- Gender participation in survey
- Voting preference trends
- Predicted election outcomes

## Files
- `PYTHON PROJECT.ipynb` – Python analysis notebook
- `2026 - ELECTION.csv` – Survey dataset
